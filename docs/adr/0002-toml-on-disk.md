# ADR 0002: TOML как формат на диске

## Статус

Accepted

## Контекст

Файлы архитектуры лежат в репозитории продукта и должны быть удобны для review,
diff и ручного чтения.

## Решение

На диске sidecar пишет только:

```text
docs/model.toml
docs/adr/*.toml
docs/flows/*.toml
```

JSON используется только как формат MCP tool call и для схем в `schemas/`.

## Последствия

- Репозиторий продукта не получает служебную БД.
- ADR и Flow читаются как обычные TOML-файлы.
- Проверки и viewer строятся из содержимого `docs/`.
