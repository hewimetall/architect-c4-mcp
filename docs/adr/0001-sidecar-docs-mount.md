# ADR 0001: один mount `docs/`

## Статус

Accepted

## Контекст

architect-c4 работает рядом с репозиторием продукта и не владеет его исходниками.
Агенту нужен простой контракт без проектов и внутреннего API управления
репозиториями.

## Решение

Продукт принимает один путь: `--docs /path/to/product/docs` или
`ARCHITECT_C4_DOCS`. Все MCP-инструменты работают с уже привязанным каталогом
`docs/`.

## Последствия

- Для запуска достаточно смонтировать `docs/`.
- История остается в git продукта.
- Публичный viewer строит ссылки `/view`, `/view/adrs`, `/view/flows`.
